=======Instructions========
Run "javac Main.java" to compile the program
Run "java Main" to run the program

=======Configurables=======
In file Time.java you can change the variable MS_PER_MIN 
to change how long the program runs for

In file Main.java you can change the minimum and maximum 
question rates.  The rates correspond to given x number of a minutes
a question should be asked.

==========Output===========
The program outputs every major event for the day including when a 
developer or person comes to work, goes to meetings, asks questions
answers questions, go/return from lunch and others.

After the day is finished the program outputs the different time 
statistics for the entire day for each developer and then 
outputs a combined TOTAL statistic for how time was spent by everyone
